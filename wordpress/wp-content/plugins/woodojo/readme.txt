== Description ==

Enhance your WordPress website with the WooThemes features that you need. Delivering top quality features & backed by a team that's done it before. Say hello to WooDojo.

[vimeo http://vimeo.com/40070437]

= FEATURES =
WooDojo has a growing list of useful features, as well as showcasing plugins developed by WooThemes.

= BUNDLED FEATURES =
WooDojo comes bundled with a collection of features to help customise your website:

* WooDojo - Tabs (a tabbed widget)
* WooDojo - Shortlinks (use your URL shortening service of choice with WordPress shortlinks)
* WooDojo - Custom Code (add custom CSS code, as well as HTML in the <head> section or above the closing </body> tag)
* WooDojo - Login Branding (rebrand the WordPress login screen with your logo)
* WooDojo - Social Widgets (display information from your social profiles via easy-to-use widgets)

== Installation ==

= Minimum Requirements =

* WordPress 3.3 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Manual installation =

The manual installation method involves downloading our feature-rich plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installationâ€™s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

= Upgrading =

Automatic updates should work a charm; as always though, ensure you backup your site just in case.
